# px_kvdb_gather.sh

## Description
Collects KVDB details for kvdb analysis. This needs to be executed from one of the PX-Node (either by ssh to the px-node or oc debug to the node)


## Usage

```bash
./pxkvdb_gather.sh
```

### Execute Using Curl
You can download and execute the script directly from GitHub using the following command:
```bash
curl -ssL https://raw.githubusercontent.com/portworx/support-external-scripts/refs/heads/main/px_kvdb_bundle/px_kvdb_gather.sh | bash
```
